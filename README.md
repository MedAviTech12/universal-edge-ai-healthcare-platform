# Universal Plug-and-Play Edge AI Platform for Multimodal Healthcare Diagnostics
This project presents a Universal Plug-and-Play Edge AI Platform designed for healthcare diagnostics. The platform enables deployment and execution of multiple AI models on a low-cost Raspberry Pi device without relying on cloud infrastructure.

The system supports multimodal healthcare data including facial images, thermal images, chest X-rays, respiratory audio recordings, and dermatological images while preserving patient privacy through local inference.
